Swift Notes is a simple, open-source Android notes app, providing a clean and user-friendly interface. Built using Jetpack Compose and Material You, it offers note-taking features with Markdown support, encryption options, and various language support. This app focuses on privacy and does not rely on proprietary services.

    Features:
    - Material You dynamic themes
    - Markdown note rendering
    - Secure notes with encryption
    - Multi-language support: English, Hindi, Bengali, Gujarati, Kannada, Malayalam, Marathi, Punjabi, Tamil, Telugu, Urdu, Odia, Assamese
    - Privacy-focused: No proprietary dependencies or tracking
